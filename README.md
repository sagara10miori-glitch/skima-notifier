# SKIMA Notifier  
SKIMA の新着商品を自動で取得し、Discord に美しい Embed で通知するシステムです。  
通知の確実性・視認性・保守性を最大限に高めた構成になっています。

---

## 🚀 機能一覧

### 🔔 通知機能
- 新着商品を Discord Webhook に自動通知
- タイトルは《  額縁スタイル  》
- 価格は「9,000円」のように桁区切り
- URL は「🔗 skima.jp/xxxx」の短縮表示
- サムネイル画像付き
- ラベル（🔥✨⭐）で重要度を表示
- 色分け（赤→青→緑→無色）で視認性UP
- 通知タイトル絵文字（📢🔔📝 + 💌）

### 👤 ユーザー管理
- `priority_users.txt`：通知対象ユーザー
- `exclude_users.txt`：除外ユーザー
- 通知対象 = 優先 − 除外
- 空行・重複・全角スペースも自動除去

### 🛡 安定性・堅牢性
- fetch：最大3回リトライ
- notify：最大2回再送＋レート制限対応
- seen.json：破損時は自動修復
- HTML構造が変わってもフェイルセーフで通知継続
- logs/notifier.log に詳細ログを保存

### ⚙ GitHub Actions
- 5分ごとに自動実行
- 自動リトライ（最大2回）
- ランダム遅延で混雑回避
- pip キャッシュで高速化
- Webhook URL は Secrets 管理
